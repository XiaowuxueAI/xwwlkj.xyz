第三步：搭建你的第一个AI自动化工作流

真正开始用AI赚钱的人，不是一次一次手动提问，而是把重复性工作自动化。比如：每天早上自动生成今日热点文案 → 自动排版 → 自动保存。这靠的是DeepSeek的API + 简单脚本。

  2. 用Python写一个20行的脚本调用API
  3. 设置定时任务，让它每天自动运行

实例：每日自动生成小红书文案

以下是你需要的基本Python脚本结构（DeepSeek可以帮你写，复制粘贴就能跑）：

import requests

API_KEY = "你的DeepSeek API Key"
prompt = "生成5条今日热点相关的小红书文案，口语化、加emoji"

response = requests.post(
    "https://api.deepseek.com/v1/chat/completions",
    headers={"Authorization": f"Bearer {API_KEY}"},
    json={"model": "deepseek-chat", "messages": [{"role": "user", "content": prompt}]}
)

print(response.json()["choices"][0]["message"]["content"])
