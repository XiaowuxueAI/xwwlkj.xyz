## 📝 写作类（12个）

### 1. 小红书种草文案
```
你是一个小红书爆款文案写手。请根据以下产品信息，写一篇小红书种草笔记：
产品：[产品名称]
卖点：[核心卖点1、卖点2、卖点3]
要求：口语化、多用emoji、分段清晰、结尾引导评论
字数：300-500字
```

### 2. 公众号深度长文
```
请以[领域]专家的身份，写一篇关于"[主题]"的深度文章。
要求：
- 开头用数据或故事吸引注意力
- 分3-5个小标题展开
- 每个观点配案例
- 结尾给出可操作建议
- 字数：1500-2000字
- 风格：专业但不枯燥
```

### 3. 短视频口播脚本
```
请为以下主题写一个60秒短视频口播脚本：
主题：[主题]
目标人群：[人群描述]
要求：
- 前3秒必须有钩子（制造悬念/反差/痛点）
- 口语化表达，像朋友聊天
- 节奏快，每句话不超过15字
- 结尾引导点赞关注
```

### 4. 朋友圈卖货文案

```
要求：
- 第1条：痛点引入型

- 第3条：限时福利型

```

### 5. 知乎高赞回答

## 🎁 加赠：3个超级提示词技巧

**技巧1：角色扮演法**
不要直接问"怎么写好文案"，而是说"你是一个有10年经验的资深文案总监"。

**技巧2：分步追问法**
第一轮：生成大纲
第二轮：展开第一部分
第三轮：优化第二部分
...

**技巧3：负向约束法**
"不要使用以下词语""不要写成XX风格"比直接说"要XX"更有效。
