# 🎬 短视频脚本模板 · 30个爆款结构

> 适用：抖音/B站/小红书/视频号
> 用法：选择题材→套对应模板→填入你的内容→直接出片

---

## 🔥 开头钩子公式（前3秒定生死）

### 模板1：反常识开头
```
"你知道吗？[公认的事实]其实是错的。"
示例："你知道吗？每天喝8杯水其实是错的。"
```

### 模板2：数字冲击
```
"[X]个人里有[X]个都不知道[某事]"
示例："10个人里有9个都不知道微信这个隐藏功能"
```

### 模板3：身份代入
```
"如果你是[某类人]，这条视频一定要看完"
示例："如果你每个月工资不到1万，这条视频一定要看完"
```

### 模板4：结果前置
```
"我用[方法]，[时间]内实现了[结果]"
示例："我用这3个AI工具，一个月副业多赚了3000块"
```

### 模板5：悬念制造
```
"接下来我要说的，可能会得罪[某类人]"
示例："接下来我要说的，可能会得罪整个培训行业"
```

## 📝 完整脚本模板

### 模板6：教程型（60秒）
```
[0-3秒] 钩子："今天教你一个[XX]的方法，学会了能[好处]"
[3-15秒] 问题展示：你现在是不是[痛点描述]？
[15-45秒] 步骤演示：
  第一步：[动作+画面]
  第二步：[动作+画面]  
  第三步：[动作+画面]
[45-55秒] 成果展示：你看，这样就[效果]
[55-60秒] 互动引导："觉得有用点个赞，下期教你[预告]"
```

### 模板7：揭秘型（45秒）
```
[0-3秒] "为什么[某现象]？今天给你讲透"
[3-15秒] 常见误区："很多人以为是[原因A]"
[15-35秒] 真实原因："实际上是因为[原因B、C、D]"
[35-42秒] 总结："所以记住，[核心观点]"
[42-45秒] 引导："转发给你身边还不知道的人"
```

### 模板8：开箱/测评型（60秒）
```
[0-5秒] 产品亮相+价格
[5-15秒] 开箱过程（快速剪辑）
[15-35秒] 功能介绍+实际使用
[35-50秒] 优缺点总结
[50-60秒] 购买建议+适合人群
```

### 模板9：Vlog故事型（90秒）
```
[0-10秒] 今日高光预告（3个最精彩画面快剪）
[10-30秒] 背景交代：时间/地点/做什么
[30-70秒] 过程展开：起→承→转→合
[70-85秒] 感悟总结：今天学到了什么
[85-90秒] 结尾：下期预告+互动
```

### 模板10：合集推荐型（60秒）
```
[0-3秒] "[X]个超好用的[XX]，最后一个绝了"
[3-50秒] 逐个介绍（每个10-15秒）
  ① [名称] - [一句话亮点] - [画面]
  ② [名称] - [一句话亮点] - [画面]
  ③ [名称] - [一句话亮点] - [画面]
  ④ [名称] - [一句话亮点] - [画面]
[50-60秒] 总结+引导收藏
```

## 🎯 带货转化脚本

### 模板11：痛点解决方案型
```
[0-5秒] 痛点场景还原："你是不是也[痛苦经历]？"
[5-15秒] 放大痛苦："试过XX没用，试过YY也不行..."
[15-35秒] 产品登场："直到我发现了[产品]"
[35-50秒] 效果展示："用了之后[具体改善]"
[50-60秒] 限时优惠："今天下单只要[价格]，链接在评论区"
```

### 模板12：对比打脸型
```
[0-5秒] "同样[场景]，[产品A]和[产品B]的差距有多大？"
[5-25秒] A的效果展示
[25-45秒] B的效果展示（明显更好）
[45-55秒] 价格对比（B更划算）
[55-60秒] "链接放评论区了，自己看"
```

### 模板13：专家推荐型
```
[0-5秒] 身份建立："做了[X]年[行业]，告诉你一个真相"
[5-20秒] 行业揭秘："市面上大多数[产品]其实都..."
[20-50秒] 推荐理由："我用了[X]款，只有这款[具体优势]"
[50-60秒] 福利："跟品牌方要了专属优惠，评论区领"
```

## 🎨 创意脚本

### 模板14：如果...会怎样型
```
[0-3秒] "如果[某条件]，会发生什么？"
[3-40秒] 几个搞笑/震惊的假设场景
[40-50秒] 揭晓答案
[50-60秒] "你还想看我测什么？评论区告诉我"
```

### 模板15：挑战型
```
[0-5秒] "挑战[任务]，失败了[惩罚]"
[5-50秒] 挑战过程（紧张/搞笑剪辑）
[50-60秒] 结果揭晓+拉票"点个赞给我力量"
```

### 模板16：一天体验型
```
[0-5秒] "体验一天[职业/身份]，太[形容词]了"
[5-20秒] 上午：初体验
[20-40秒] 下午：渐入佳境/意外频发
[40-55秒] 晚上：总结感受
[55-60秒] 互动："下期想看什么？"
```

## 📱 平台适配

### 模板17：抖音暴款模板
```
[0-1秒] 炸裂画面/表情
[1-3秒] 悬疑文案："接下来..."
[3-45秒] 快节奏信息输出（每2-3秒切一个画面）
[45-60秒] 评论引导："你中了几条？"
音乐：热门BGM + 踩点剪辑
```

### 模板18：B站深度内容模板
```
[0-10秒] 片头动画+本期主题
[10-60秒] 背景介绍：为什么聊这个
[1-5分钟] 核心内容：分3-5个点展开
[5-8分钟] 深度分析：个人观点+数据支撑
[8-10分钟] 总结+下期预告
弹幕互动：中间穿插"觉得对扣1，不对扣2"
```

### 模板19：小红书短视频模板
```
[0-2秒] 精美封面+标题文字
[2-5秒] "分享一个[XX]的小技巧"
[5-40秒] 步骤演示（竖屏9:16，字幕要大）
[40-45秒] Before/After对比
[45-50秒] "关注我，每天分享[XX]"
```

### 模板20：视频号模板
```
[0-5秒] 亲切问候："朋友们好，今天聊聊[话题]"
[5-50秒] 娓娓道来（真人出镜/口播为主）
[50-60秒] 金句收尾+转发引导
风格：温暖、走心、有价值
```
