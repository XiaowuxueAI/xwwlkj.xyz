# 🤖 普通人用AI提升效率的5个实操技巧

> 不需要技术背景，不需要付费工具。看完就能用，用了就有效。

---

## 技巧1：把AI当"会干活的实习生"，不是"会聊天的网友"

**90%的人用AI的方式是错的。** 他们打开DeepSeek/豆包/千问，像跟朋友聊天一样随便问几句，得到随便的回答，然后觉得"AI也就那样"。

**正确做法：** 把AI当成一个**聪明但需要明确指令的实习生**。

你说得越具体，它做得越好。

❌ 错误："帮我写个文案"
✅ 正确："我是淘宝卖家，卖手工皂。帮我写一个详情页卖点文案。产品特点：天然成分、孕妇可用、泡沫丰富。目标是25-35岁女性。要突出安全性和使用体验。字数200字以内。"

---

## 技巧2：给AI看"参考答案"，让它照葫芦画瓢

AI最擅长的事不是创造，而是**模仿**。

如果你想让AI写出和你风格一致的东西，给它看一个例子。

**操作步骤：**
1. 找到一段你喜欢的文案/文章/代码
2. 粘贴到对话框，前面加上："参考以下风格，写一段关于[你的主题]的[文案/文章]："
3. AI会模仿那个风格和结构来写

这个技巧适用于：文案写作、邮件回复、周报月报、朋友圈文案。

---

## 技巧3：用"分步骤指令"替代"一句话指令"

人类喜欢一步到位，但AI更适合分步骤来。

**示范：写一篇小红书笔记**

❌ 一步到位："帮我写一篇关于咖啡的小红书笔记"
✅ 分步骤：
1. 先让AI列出5个吸引人的标题选项
2. 选最好的那个标题
3. 基于标题，让AI写正文（第一段：痛点引入 → 第二段：产品介绍 → 第三段：使用感受 → 第四段：购买建议）
4. 让AI给正文加上emoji和分段
5. 最后让AI推荐5个标签

每一步都检查调整，最后出来的成品质量远超一次性生成。

---

## 技巧4：建立你自己的"提示词模板库"

重复写同样的提示词是最大的时间浪费。

**花10分钟，一次性建好5个常用模板，以后直接复制改关键词就行。**

模板1：工作周报
```
帮我写一份周报。
本周完成：[填入具体工作]
遇到的问题：[填入困难]
下周计划：[填入安排]
语气：专业但不死板。500字以内。
```

模板2：朋友圈产品文案
```
帮我写一条朋友圈推广文案。
产品：[名称]
卖点：[3个核心卖点]
风格：自然接地气，不要太广告感。
加上合适的emoji。
```

模板3：邮件回复
```
帮我回复这封邮件：
[粘贴邮件内容]
要求：礼貌、简洁。表达感谢。明确下一步行动。
```

## 技巧5：用AI处理你不擅长的事

AI不是万能的，但它在你**不擅长的事情上**性价比最高。

**真实场景对照表：**

| 你很擅长 | 你不太行 | 用AI搞定的 |
|---------|---------|-----------|
| 产品知识 | 文案写作 | ✅ AI写文案 |
| 业务运营 | 数据分析 | ✅ AI做表 |
| 沟通能力 | 外语翻译 | ✅ AI翻译 |
| 创意想法 | PPT制作 | ✅ AI框架 |
| 行业经验 | 简历优化 | ✅ AI润色 |

**核心原则：** 你专注做你擅长的，AI补齐你不擅长的。这就是效率。
