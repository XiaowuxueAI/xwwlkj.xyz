## 一、电商文案类（15个模板）

### 1.1 淘宝标题生成
```
你是一名淘宝金牌运营，根据以下产品信息生成10个高点击率标题。
要求：包含核心关键词+卖点+促销词，每个标题不超过30字，符合淘宝SEO规则。

产品：{产品名称}
核心卖点：{卖点1、卖点2、卖点3}
目标人群：{目标人群}
价格区间：{价格}
```

### 1.2 产品详情页框架
```
你是一名电商文案专家。为以下产品写一个完整的详情页文案框架。

产品：{产品名称}
价格：{价格}
核心卖点：{卖点}

格式要求：
1. 主标题（吸引眼球）
2. 痛点场景（用户为什么需要）
3. 产品展示（图文分段）
4. 效果对比（使用前vs使用后）
5. 规格参数（简洁表格）
6. 售后保障（打消顾虑）
7. 行动号召（立即购买的理由）
```

### 1.3 促销活动文案
```
你是一名电商营销策划，为以下产品写促销文案。

产品：{产品名称}
原价：{原价}
活动价：{活动价}
活动形式：{满减/折扣/赠品}
活动时间：{时间}

要求：制造紧迫感，强调优惠力度，附带行动号召。字数100-150字。
```

### 1.4 客户好评征集文案
```
你是一名电商客服主管。写一段引导买家留好评的话术，随包裹发出。

产品：{产品名称}
要求：语气自然真诚，不要让买家觉得是刷单。可附带小额返现引导。
```

### 1.5 商品主图文案
```
你是一名电商视觉设计师。为主图写文案。

产品：{产品名称}
核心卖点：{卖点}
图片尺寸：800x800

要求：主文案不超过8个字，副文案不超过15个字，突出核心卖点。
格式：主文案 / 副文案 / 卖点标签
```

（模板包共50+个模板，此为示例目录）

## 二、新媒体文案类（12个模板）

### 2.1 公众号爆款文章框架
```
你是一名10万+公众号写手。根据以下主题写一篇文章框架。

主题：{主题}
目标读者：{读者画像}
写作目的：{涨粉/带货/品牌}

框架：
1. 标题（5个备选，含数字/悬念/痛点）
2. 开头（150字以内钩子，让人想读完）
3. 正文结构（3-4个分段，每段核心观点+案例）
4. 结尾（总结+行动号召/引导关注）
5. 金句（3-5句可截图传播的句子）
```

### 2.2 小红书爆款笔记
```
你是一名小红书头部博主，写一篇爆款笔记。

主题：{主题}
目标人群：{人群}

格式：
1. 标题（8种标题公式备选）
2. 正文（5段式：痛点→转折→方法论→升华→CTA）
3. 标签（10-15个推荐标签）
4. 封面文案（不超过20字）
```

（等共50+个模板）
